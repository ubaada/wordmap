Word Voyage

See how closely AI relates words to each country.

About

This project was created by Ubaada

How does it work?

It uses word-embeddings generated by an AI model to represent words as vectors. Through training, the model has learned to associate numbers with words in a way that words with similar meanings have vectors that are closer numerically. For example, "car" and "van" have similar vectors but "car" and "carpet" do not. The script then uses a similarity measure to compare the vectors of the input word and the vectors of the name of each country. The darker the color, the more similar the word is associated with the country.

Specification

Model: text-embedding-ada-002
Similarity measure:

  • Cosine Similarity
  • Euclidean Similarity

Tools:
  • Plotly JS
  • Vector operations in vanilla PHP
